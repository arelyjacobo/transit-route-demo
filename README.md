# Prague Transit Route Demo

A simple web application that simulates a transit route planner inspired by Prague’s public transportation system.

The app allows users to select two stations and generate a route between them. The interface displays route information and visualizes the route on an interactive map.

### Features

- Interactive route finder for selected Prague metro stations
- Dynamic route visualization on an interactive Leaflet map
- Animated metro-style route rendering
- Intermediate station indicators along the route
- Start and destination station markers
- Station swap feature to instantly reverse routes
- Multilingual interface (English, Czech, German, Spanish)
- Light and dark theme toggle

### Technologies

- HTML
- CSS
- JavaScript
- Leaflet.js (for the interactive map)
- OpenStreetMap
